## Features
- Admin Mode:
  - Add Student Record
  - Remove Student Record
  - View Student Record
  - View All Records
  - Edit Admin Password
  - Edit Student Grade
  - Quit The System

- Student Mode:
  - View Student Record
  - Edit His Password
  - Edit His Name
  - Quit The System
